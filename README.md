# 🎬 Movie Recommendation System

A Machine Learning-based Movie Recommendation System built with Python and Streamlit that suggests similar movies using content-based filtering and cosine similarity. The app fetches movie posters dynamically using the TMDB API and provides an interactive, user-friendly interface for discovering movies.

## 🚀 Features

* Movie recommendations based on similarity
* TMDB API integration for posters
* Interactive Streamlit UI
* Fast recommendation engine

## 🛠️ Technologies Used

* Python
* Pandas
* Scikit-learn
* Streamlit
* TMDB API

## ▶️ Run Locally

```bash
pip install -r requirements.txt
streamlit run app.py
```
